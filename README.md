
# Leaflet 東京エリア マップ表示サンプル

Leaflet.jsとOpenStreetMapを使用した、シンプルで軽量なWeb地図表示アプリケーションです。
初期表示時に東京エリア（東京都庁周辺）を中心に配置し、東京駅にピン（マーカー）を立ててポップアップを表示します。

## 主な機能

* **オープンデータ地図の利用**: OpenStreetMapのタイルレイヤーを使用しているため、APIキーの取得が不要で即座に動作します。
* **初期位置の設定**: 東京エリア（緯度: 35.6895, 経度: 139.6917）を中心に、広域が見渡せるズームレベル（12）で起動します。
* **マーカー＆ポップアップ**: 東京駅の位置にデフォルトでマーカーを配置し、クリック（または自動）で詳細ポップアップを表示します。
* **レスポンシブ対応**: スマートフォンやタブレット、PCなど、あらゆる画面サイズにおいてフルスクリーンで地図がフィットします。

## 使用している技術・ライブラリ

本プロジェクトは以下のCDNライブラリを組み込んで構築されています：

* **Leaflet.js (v1.9.3)** - 軽量なオープンソースのJavaScript地図ライブラリ
* **jQuery (v3.7.1)** - ドム操作・スクリプト支援
* **Bootstrap (v5.2.2 & v3.0.0 Glyphiccons)** - スタイリングとコンポーネント
* **FontAwesome (v6.2.0)** - アイコン表示用
* **Leaflet.awesome-markers (v2.0.2)** - カスタマイズ可能なマップマーカー用プラグイン

## 使い方（クイックスタート）

1.  本リポジトリにある `index.html`（または該当のHTMLファイル）をローカル環境にダウンロードします。
2.  ダウンロードしたHTMLファイルを、Google ChromeやSafariなどのブラウザで直接開きます。
3.  インターネットに接続されていれば、自動的に東京の地図が描画されます。

## 開発・カスタマイズ方法

地図の中心地やズーム倍率を変更したい場合は、コード内の以下の数値を編集してください。

### 1. 中心座標とズーム倍率の変更
`index.html` 内の `L.map` の設定箇所を変更します。
```javascript
center: [35.6895, 139.6917], // [緯度, 経度] を変更
zoom: 12,                    // 0〜19の間で数値を変更（大きいほど拡大）
2. マーカー（ピン）の変更・追加
コード末尾にあるマーカーの座標やポップアップの文字を変更、または複製することでピンを増やせます。

JavaScript
// [緯度, 経度] を変更して別の場所にピンを立てる
var marker = L.marker([35.6812, 139.7671]).addTo(map_befff7fd11199784079e61338f4d633f);
// ポップアップ内のテキスト（HTMLタグ使用可）を変更
marker.bindPopup("<b>任意のタイトル</b><br>説明文").openPopup();
